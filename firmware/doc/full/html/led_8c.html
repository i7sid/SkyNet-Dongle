void skynet_led_blink_passive ( uint32_t  ms)

Lights up the LED for a given amount of time.

Parameters
msTime in milliseconds the LED will be on.

The specified amount of time will be waited passively, so the code execution will not be delayed.

Warning
This can lead to unexpected behaviour, if you call this function again before the last execution was finished.
