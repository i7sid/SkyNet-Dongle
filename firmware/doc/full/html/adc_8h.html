void adc_activate ( void  )

Activate ADC measurement circuit.

Switches the MOSFETs so that the measurement can be done. In this state the measurement circuit consumes current.

void adc_deactivate ( void  )

Deactivate ADC measurement circuit.

Switches the MOSFETs so that the measurement can not be done. In this state the measurement circuit consumes no current.
