Macro Definition Documentation

#define MAX_DELAYED_EVENTS   (32)

The maximum amount of events that can be enqueued in parallel.

This value is used to instantiate the queue, so be aware that a high value may consume unnecessarily much memory.

void msDelay ( uint32_t  ms)

Blocks for a specific amount of time (passively).

Parameters
msMilliseconds to block.

CPU can sleep in this time or handle incoming interrupts.

Warning
Do not use this function in interrupt handlers! This will break your code! Use register_delayed_event() or events_enqueue() for delaying in ISRs.
void msDelayActive ( uint32_t  ms)

Blocks for a specific amount of time (actively).

Parameters
msMilliseconds to block.

CPU can handle incoming interrupts.

Warning
Do not use this function in interrupt handlers! This will prevent regular code and possibly other interrupt routines from being executed. Use register_delayed_event() or events_enqueue() for delaying in ISRs.
