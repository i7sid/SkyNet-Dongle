bool charger_status1 ( void  )

Returns the value of STAT1.

Basically this is low / false if battery is charging or an error occured. For exact status see datasheet: http://www.microchip.com/wwwproducts/Devices.aspx?dDocName=en536670

Returns
true: Shutdown, Charge Complete, No Battery Present, No Input Power Present; false: Charging, Fault

Returns the value of STAT2.

Returns
true: Shutdown, Charging, No Battery Present false: Charge Complete, Fault
